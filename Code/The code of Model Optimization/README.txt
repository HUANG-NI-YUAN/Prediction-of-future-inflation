1. If you have anaconda or jupyterbook installed on your computer, you can directly open the ".ipynb" file to view it, if not, you can open the ".mhtml" file to browse the code.
	1.1 If you want to run the code properly, you should have the numpy library, pandas library, matplotlib library, statsmodel library, sklearn library and xgboost library installed on your computer. If these libraries are already installed but not running, check if the version is too low and try to update them.
2. data folder is the source of data, where "data_raw" is the raw data and "data" is the data after some simple processing.
3. results folder holds the images generated by the program.
